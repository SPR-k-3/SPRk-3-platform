# Security Policy

## Purpose
SPR{K}3 Scanner is a defensive security tool designed to help developers identify and fix vulnerabilities in ML/AI code.

## Responsible Use
This tool should only be used for:
- Scanning your own code
- Authorized security research
- Bug bounty programs with permission
- Improving software security

## Reporting Vulnerabilities
If you find vulnerabilities using this tool:
1. Report them responsibly to project maintainers
2. Allow reasonable time for fixes (90 days standard)
3. Do not exploit or publicize unfixed vulnerabilities

## Ethical Guidelines
- Never use for unauthorized access
- Follow all applicable laws and regulations
- Respect intellectual property
- Prioritize user safety and privacy

## Vulnerability Disclosure
Contact: security@sprk3.com
Timeline: 90-day responsible disclosure
